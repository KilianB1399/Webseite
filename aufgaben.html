Aufgabe 1: Erste HTML-Seite erstellen

Erstelle eine einfache HTML-Seite mit der grundlegenden Struktur und einem Titel.

Lösung:

</head>

    <h1>Willkommen auf meiner Webseite</h1>
    <p>Dies ist mein erster Versuch mit HTML.</p>
</body>
</html>

Aufgabe 19: Dropdown-Menü mit JavaScript

Erstelle ein Dropdown-Menü, das mit einem Klick ein- und ausgeblendet wird.

Lösung:

<!-- HTML -->
<div class="dropdown-container">
    <button onclick="toggleDropdown()">Dropdown anzeigen</button>
    <div class="dropdown" id="myDropdown">
        <a href="#">Link 1</a>
        <a href="#">Link 2</a>
        <a href="#">Link 3</a>
    </div>
</div>

<!-- JavaScript -->
<script>
function toggleDropdown() {
    var dropdown = document.getElementById('myDropdown');
    if (dropdown.style.display === 'block') {
        dropdown.style.display = 'none';
    } else {
        dropdown.style.display = 'block';
    }
}
</script>

<!-- CSS -->
/* styles.css */
.dropdown-container {
    position: relative;
    display: inline-block;
}

.dropdown {
    display: none;
    position: absolute;
    background-color: #f1f1f1;
    min-width: 160px;
    box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2);
    z-index: 1;
}

.dropdown a {
    color: black;
    padding: 12px 16px;
    text-decoration: none;
    display: block;
}

.dropdown a:hover {
    background-color: #ddd;
}
</style>

Aufgabe 20: Kontaktformular mit JavaScript validieren

Erstelle ein Kontaktformular mit JavaScript, das die Eingaben überprüft, bevor es gesendet wird.

Lösung:

<!-- HTML -->
<form id="contactForm">
    <input type="text" id="contactName" placeholder="Name"><br><br>
    <input type="email" id="contactEmail" placeholder="E-Mail"><br><br>
    <textarea id="contactMessage" placeholder="Nachricht"></textarea><br><br>
    <button type="button" onclick="validateForm()">Senden</button>
</form>

<!-- JavaScript -->
<script>
function validateForm() {
    var name = document.getElementById('contactName').value;
    var email = document.getElementById('contactEmail').value;
    var message = document.getElementById('contactMessage').value;
    if (name == "" || email == "" || message == "") {
        alert("Bitte füllen Sie alle Felder aus.");
    } else {
        alert("Vielen Dank für Ihre Nachricht!");
    }
}
</script>
