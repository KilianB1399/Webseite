HTML & CSS Aufgaben für Anfänger

Aufgabe 1: Erste HTML-Seite erstellen

Erstelle eine einfache HTML-Seite mit der grundlegenden Struktur und einem Titel.

Lösung:

<!DOCTYPE html>
<html>
<head>
    <title>Meine erste Webseite</title>
</head>
<body>
    <h1>Willkommen auf meiner Webseite</h1>
    <p>Dies ist mein erster Versuch mit HTML.</p>
</body>
</html>

Aufgabe 2: Weitere HTML-Elemente hinzufügen

Füge eine ungeordnete Liste, einen Hyperlink und ein Bild zu deiner HTML-Seite hinzu.

Lösung:

<ul>
    <li>HTML lernen</li>

Aufgabe 3: Einführung in CSS

Erstelle eine externe CSS-Datei und verknüpfe sie mit deiner HTML-Datei, um grundlegende Stile hinzuzufügen.

Lösung:

<!-- Verknüpfung der CSS-Datei -->
<head>
    <link rel="stylesheet" href="styles.css">
</head>

<!-- Inhalt der styles.css -->
/* styles.css */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}

h1 {
    color: #2c3e50;
}

p {
    font-size: 18px;
}

Aufgabe 4: Layout mit CSS

Erstelle eine Box mit CSS, um das CSS-Box-Modell zu verstehen.

Lösung:

<!-- HTML -->
<div class="box">
    <h2>Eine Box</h2>
    <p>Dies ist eine einfache Box mit ein wenig Text darin.</p>
</div>

<!-- CSS -->
/* styles.css */
.box {
    width: 300px;
    padding: 20px;
    margin: 20px auto;
    background-color: #fff;
    border: 1px solid #ccc;
    box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}

.box h2 {
    margin-top: 0;
}

Aufgabe 5: Responsive Design mit CSS

Verwende Media Queries, um deine Webseite für verschiedene Bildschirmgrößen anzupassen.

Lösung:

<!-- CSS -->
/* styles.css */
@media (max-width: 600px) {
    .box {
        width: 100%;
        padding: 10px;
    }
}
